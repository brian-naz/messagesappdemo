# Messaging App Demo

To get started:
<a href="https://messagesappdemo.vercel.app/">Demo</link>

### Setup:

Clone the repo:
```bash
git clone https://github.com/brian-naz/messagesappdemo.git
```
Go to the directory and simply type:
```bash
npm install
```
To run:
```bash
npm start
```
