# Messaging App Demo

An barebones iOS Messaging App I made in react
It doesn't support any servers, just based on a fixed preset of responses

### Setup:

Clone the repo:
```bash
git clone https://github.com/brian-naz/messagesappdemo.git
```
Go to the directory and simply type:
```bash
npm install
```
To run:
```bash
npm start
```
